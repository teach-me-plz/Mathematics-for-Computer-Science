# rem 이란?

| 연산                        | 값     | 설명                                      |
| --------------------------- | ------ | ----------------------------------------- |
| $-7 \bmod 3$                | **2**  | 항상 $0 \le r < b$                        |
| $-7 \mathbin{\text{rem}} 3$ | **-1** | 나눗셈 결과를 0에 가장 가까운 정수로 버림 |
